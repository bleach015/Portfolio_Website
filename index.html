SortOut

This is a senior project working in duo.
SortOut is an education game based iOS application developed using Swift.
It use the technique of remembering vocabulary using sentence arrangements.
It will randomize some words and what you need to do is arranging them into a meaningful sentence. If the sentence is long and correct you get 3 stars, otherwise you get 2 stars, 1 star, and 0 star. It also has a built-in dictionary which will show only the randomized word if you not sure what they mean. You can add words in the dictionary to the vocab list for later memorization.
The application uses Natural Language Processing to detect lemmatization and use GoogleML for in-device text translation (offline).
The application was developed using Swift language.
